# Bottelegram22

## Descrição
Bot para Telegram que responde a comandos de atendimento ao cliente, compras e vendas.

## Funcionalidades
- Responde a comandos básicos (/start, /help).
- Integração com Dialogflow para respostas automáticas.
- Suporte a vendas e atendimento.

## Comandos
- `/start` - Iniciar bot.
- `/ajuda` - Mostrar ajuda.
- `/produto` - Informações sobre produtos.
